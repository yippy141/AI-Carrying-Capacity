# Visual Components

V0 visual components for the AI Conversion Atlas.

Current components:

- `ConversionFunnel`: five-layer capability-to-outcomes funnel.
- `CountryCompare`: country layer states without a composite score.
- `BottleneckFingerprint`: qualitative watch-state matrix.
- `SectorHeatmap`: manufacturing evidence-state heatmap.
- `ScenarioSimulator`: local JSON, client-side hypothesis pattern matcher.
- `SourceBackedClaimCard`: claim, caveat, source ID, and review-state display.
- `UncertaintyLegend`: shared evidence-label legend.

These components read local JSON and must keep missing values, official claims,
qualitative coding, estimates, and observed evidence visually distinct.
