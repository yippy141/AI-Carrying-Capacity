# Deep Research Reports

Save reports here using:

```text
YYYY-MM-DD-topic.md
```

Examples:

```text
2026-06-27-literature-review.md
2026-06-27-existing-indices.md
2026-06-27-china-ai-plus.md
2026-06-27-manufacturing-robotics.md
```

For each report, create staged extraction files in `research/source-register/`.
